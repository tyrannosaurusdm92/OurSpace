# Docs Folder Map

- `audits/source-repo-audit.md` documents each uploaded repo and why code was or was not used.
- `licenses/license-notes.md` summarizes observed licenses and reuse decisions.
- `manifests/module-manifest.json` lists shipped module files and features.
- `manifests/source-repo-manifest.json` contains machine-readable source repo audit metadata.
- `readmes/INTEGRATION.md` explains how to add the module to an OurSpace page.
- `readmes/BACKEND_CONTRACT.md` explains the expected Google Apps Script request/response shapes.
- `readmes/UNUSED_CODE.md` explains how unused code was handled.
