# Unused Code Documentation

The uploaded repositories were reviewed as source material, but their application code was not copied into the final module.

Reasons code was not bundled:

- several repos are complete full-stack apps with server/database/framework requirements that would bloat the site
- GPL projects were not copied to avoid pulling GPL obligations into the site module
- PolyForm Noncommercial code was not copied because it is restrictive for reuse
- old tutorial/blog/content repos did not match the requested embedded journaling module
- the requested OurSpace feature can be fulfilled with a smaller, dependency-free module

What was kept:

- folder/category/tag organization patterns
- local-first safety saving
- readable entry view separate from editing
- import/export/download workflows
- backend sync adapter concept

What was shipped:

- new standalone HTML/CSS/JS written for this task
- no copied third-party app source code
- no unused repository code folders
