Local-first support scanner

Health & Behavior Reference Scanner

Scan chatbot conversations, app-owned files, and attached documents for direct needs, possible implied needs, DBT skill options, physical-health support notes, and citeable document references.

Privacy: this static version runs in your browser. Imported files and pasted conversations stay local unless you connect it to your own app backend.

Safety boundaries

This scanner is for organization, reflection, accessibility support, and chatbot reference assistance. It does not diagnose, prescribe, or replace medical, mental-health, emergency, or crisis care.

If a scan suggests imminent danger, self-harm risk, violence risk, medical emergency, or inability to stay safe, use emergency services or a crisis line instead of relying on a chatbot.

1. Attach program files or documents

Add the files the attached program possesses. Text, JSON, Markdown, HTML, CSV, and plain logs work directly. For DOCX/PDF folders, use the included Python helper to create an import package.

2. Paste chatbot conversation

Paste a transcript, message array, support chat, journaling text, or chatbot memory. The scanner looks for direct words and cautious “between the lines” signals.

3. Scan results

Run a scan to see signals, document references, and a chatbot-ready support prompt.

4. Chatbot attachment instructions

  1. Let the host app pass its owned files, attached documents, and conversation transcript into the scanner.
  2. Run the scan before the chatbot answers, or as a helper panel beside the chat.
  3. Send the generated prompt plus the citeable reference pack to the chatbot.
  4. Tell the chatbot to label uncertain findings as “possible,” use citations, and never diagnose.
