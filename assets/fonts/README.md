# Font placeholders

Font files are intentionally not bundled. Keep your separate font files and drop them into these folders using the filenames already referenced by `assets/css/ourspace-auth.css`.

Expected placeholders:
- `love-ya-like-a-sister/LoveYaLikeASister.ttf`
- `morris-roman/MorrisRoman-Black.ttf`
- `morris-roman/MorrisRomanAlternate-Black.ttf`
