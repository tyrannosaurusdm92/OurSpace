# Universal Website Color Law Facelift

The facelift adds a JavaScript theme generator in `app/theme-law.js` and color-law CSS overrides in `app/styles.css` and `app/messenger-plugin.css`.

Readable interface elements use only Lightest/Darkest pairings for buttons, navigation, forms, text, cards, panels, and messenger controls. Light, Medium, and Dark palette colors are reserved for borders, dividers, scrollbars, outlines, hover glow, and decorative accents.

Fonts used through Google Fonts CSS imports:

- Title: Metamorphous
- Heading 1: Luckiest Guy
- Heading 2: Original Surfer
- Heading 3: Ribeye
- Heading 4: DynaPuff
- Heading 5: Rum Raisin

A “Reroll theme” button in the header generates a new legal color-law theme and saves it in localStorage.
