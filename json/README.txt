OurSpace JSON structure

All active site data JSON is centralized under this one top-level /json folder.

Runtime folders:
- currency/
- dbt/
- games/
- jasper/
- william/
- onyx/

Profile module filenames:
- jasper/jasper_tasks.json
- jasper/jasper_store.json
- jasper/jasper_schedule.json
- jasper/jasper_calendar.json
- jasper/jasper_hobby.json
- william/william_tasks.json
- william/william_store.json
- william/william_schedule.json
- william/william_calendar.json
- william/william_hobby.json

Reference/admin/site JSON that is not active games, DBT, Onyx, or personal profile history is under json/docs/.
