# Onyx Emotional Support Animal Merge

This build keeps Jasper's Squishy Care site intact and adds Lord Onyx Blepman into the helper bot as an emotional support animal companion.

Added without removing existing features:
- Onyx mood images and personality data under `assets/onyx-moods/`, `js/onyx-personality-data.js`, and `data/onyx_personality_reference.json`.
- Onyx companion mood panel, quick actions, snack/comfort/pep/lore responses, and idle behavior inside the existing DBT chatbot.
- Onyx lore display on the Serotonin page.
- Caregiver work planning placeholder modules on the Calendar and Today's Schedule pages, with local browser saving for notes.
- Navigation changed into a dropdown while the currency tracker remains visible in the nav bar and live strip.

Existing games, DBT catalogues, exports, local saves, gallery uploads, store, away watcher, diary cards, journaling, and mobile responsiveness were preserved.
